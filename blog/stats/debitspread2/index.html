Following up on our previous discussion about monitoring existing positions and scanning for new ones, the natural next question for an options trader is "Does this strategy actually work over time?" The scripts we looked at before are excellent for real-time decision-making, but they are silent on the most critical element. Statistical significance.

This led me to build a Realistic Options Strategy Backtester. Its purpose is not to find a single good trade for next week, but to determine if a specific trade idea has a genuine, repeatable edge, or alpha, across hundreds of past events.

While the scanner from our last talk evaluates what's available in the market now, the backtester asks a more profound question. If we had systematically applied this strategy over the last several years, would we have made money, and was that profit more than just luck or market exposure?

The core of the new script is a simulation that replays history, focusing on predictable event cycles like earnings reports. For a stock like Apple (AAPL), it fetches years of price data and historical earnings dates. Then, for each earnings announcement in the past, it executes a defined strategy, in this case, selling a call spread just before the event. Crucially, the script operates with "amnesia" at each step; it only uses data that would have been available before the trade was placed, meticulously avoiding the fatal flaw of look-ahead bias.

But a simple profit calculation is not enough. The market is noisy. A strategy can be profitable by random chance. To separate signal from noise, the backtester incorporates several layers of statistical rigor that go far beyond a simple P/L statement:

  1. Benchmarking & Alpha Detection. it doesn't just measure absolute profit. It compares the strategy's returns to a benchmark like the SPY ETF. Using a CAPM regression, it calculates alpha, the excess return not explained by the market's movement. A positive, statistically significant alpha (with a p-value < 0.05) is the holy grail, suggesting a genuine edge. A high return with zero alpha likely means your strategy is just a complicated way of buying the market.

  2. Reality Checks with Costs: Every trade is burdened with realistic friction. The model deducts commissions and, more importantly, accounts for the bid-ask spread by giving you a worse fill price. This often turns theoretically profitable strategies into net losers.

  3. Robustness Testing: A strategy that only works with one perfect set of parameters is probably a historical fluke. This backtester stress-tests the idea by running it through different scenarios, "Ultra-Conservative," "Aggressive," etc., varying the spread width, days to expiration, and costs. If the edge evaporates with slightly different parameters, it wasn't a robust edge to begin with.

So, does selling call spreads around earnings generate alpha? The answer, as with most things in finance, is "it depends." The backtester's value is in giving you a definitive, data-driven answer for a specific stock and strategy. You might find that for a stable, mega-cap stock, the strategy has a slightly positive but insignificant alpha, not worth the risk. Conversely, you might discover that for a certain type of high-volatility stock, there is a statistically significant, repeatable inefficiency to exploit.

The final output is more than just a report; it's an actionable trading plan. Based on the historical analysis, it tells you the optimal strike selection, position size, and target premium for the next earnings event, translating years of data into a concrete ticket you could place on Robinhood.

In essence, this backtester completes the loop. The scanner tells you what you could do today. The monitor tells you how your existing position is doing. But the backtester is what allows you to answer the ultimate question: "Should I be placing this trade at all?" It moves you from trading on gut feeling and isolated examples to making decisions grounded in statistical evidence.

def fetch_price_data(ticker, start, end):
    raw = yf.download(
        ticker,
        start=start,
        end=end,
        progress=False,
        auto_adjust=False
    )

    if raw is None or len(raw) == 0:
        raise ValueError("No data returned from yfinance")

    px = None

    if 'Adj Close' in raw.columns:
        px = raw['Adj Close']
    elif 'Close' in raw.columns:
        px = raw['Close']
    else:
        if isinstance(raw.columns, pd.MultiIndex):
            for col in raw.columns:
                if 'Adj Close' in str(col) or 'Close' in str(col):
                    px = raw[col]
                    break

    if px is None:
        for col in raw.columns:
            if pd.api.types.is_numeric_dtype(raw[col]):
                px = raw[col]
                break

    if px is None:
        raise KeyError(f"Could not extract price series for {ticker}")

    if isinstance(px, pd.DataFrame):
        px = px.iloc[:, 0]

    px = pd.Series(px.values, index=px.index, name='price')
    px = px.dropna()

    if px.empty:
        raise ValueError("Price series is empty after extraction")

    df = pd.DataFrame({'price': px})
    df['returns'] = np.log(df['price']).diff()
    return df.dropna()

def get_earnings_dates(ticker, start, end):
    sym = yf.Ticker(ticker)
    try:
        cal = sym.get_earnings_dates()
        if cal is None or len(cal) == 0:
            px = fetch_price_data(ticker, start, end)
            dates = px.index[::21]
            return [d.strftime('%Y-%m-%d') for d in dates]

        cal = cal.dropna(subset=['EPS Estimate'])
        cal_dates = cal.index.tz_localize(None)

        start_dt = pd.to_datetime(start)
        end_dt = pd.to_datetime(end)

        cal = cal[(cal_dates >= start_dt) & (cal_dates <= end_dt)]

        if len(cal) == 0:
            px = fetch_price_data(ticker, start, end)
            dates = px.index[::21]
            return [d.strftime('%Y-%m-%d') for d in dates]

        return [d.strftime('%Y-%m-%d') for d in cal.index]
    except Exception as e:
        print(f"Warning: Could not fetch earnings dates: {e}")
        px = fetch_price_data(ticker, start, end)
        dates = px.index[::21]
        return [d.strftime('%Y-%m-%d') for d in dates]

def get_next_earnings_date(ticker):
    sym = yf.Ticker(ticker)
    try:
        earnings_cal = sym.get_earnings_dates()
        if earnings_cal is None or earnings_cal.empty:
            return "No earnings data available"

        today = pd.Timestamp.now().normalize().tz_localize(None)
        earnings_dates = earnings_cal.index.tz_localize(None)
        future_earnings = earnings_cal[earnings_dates > today]

        if not future_earnings.empty:
            next_earnings = future_earnings.index[0]
            return next_earnings.strftime('%Y-%m-%d')
        else:
            return "No upcoming earnings dates found"

    except Exception as e:
        return f"Error fetching earnings date: {e}"

def bs_call(S, K, T, r, vol):
    if T<=0:
        return max(S-K,0)
    d1 = (np.log(S/K) + (r + 0.5*vol*vol)*T)/(vol*np.sqrt(T))
    d2 = d1 - vol*np.sqrt(T)
    return S*norm.cdf(d1) - K*np.exp(-r*T)*norm.cdf(d2)

def price_call_spread(S, K1, K2, T, r, vol):
    premium = bs_call(S,K1,T,r,vol) - bs_call(S,K2,T,r,vol)
    return max(premium, 0.01)

def run_scenario_backtest(params):
    global price_data, events

    S = price_data['price']
    results = []

    spread_width = params['spread_width']
    days_to_exp = params['days_to_expiration']
    min_history = params['min_history_days']
    commission = params['commission_per_contract']
    bid_ask_spread = params['bid_ask_spread_pct']

    for evt in events:
        try:
            evt_dt = pd.to_datetime(evt)
        except:
            continue

        if evt_dt in S.index:
            idx = S.index.get_loc(evt_dt)
        else:
            days_diff = np.abs(S.index - evt_dt)
            closest_idx = days_diff.argmin()
            idx = closest_idx
            evt_dt = S.index[closest_idx]

        if idx < min_history or idx + days_to_exp >= len(S):
            continue

        hist = price_data.iloc[idx-min_history:idx]
        S0 = S.iloc[idx]
        S_exp = S.iloc[idx+days_to_exp]

        vol_estimate = np.sqrt(hist['returns'].var() * 252)
        vol_estimate = max(min(vol_estimate, 1.0), 0.15)

        K1 = round(S0)
        K2 = K1 + spread_width

        T = days_to_exp/252

        theoretical_premium = price_call_spread(S0, K1, K2, T, 0, vol_estimate)

        premium_after_costs = theoretical_premium * (1 - bid_ask_spread) - (2 * commission)

        min_acceptable_premium = 0.10
        strike_adjustment = 0

        while premium_after_costs < min_acceptable_premium and strike_adjustment < 10:
            strike_adjustment += 1
            K1 = round(S0 * (1 + 0.01 * strike_adjustment))
            K2 = K1 + spread_width
            theoretical_premium = price_call_spread(S0, K1, K2, T, 0, vol_estimate)
            premium_after_costs = theoretical_premium * (1 - bid_ask_spread) - (2 * commission)

        if premium_after_costs < min_acceptable_premium:
            continue

        premium_after_spread = max(theoretical_premium * (1 - bid_ask_spread), 0.01)
        commission_cost = 2 * commission
        net_premium_received = premium_after_spread - commission_cost

        payoff = min(max(S_exp - K1, 0), spread_width)
        pl = payoff - net_premium_received

        capital_at_risk = spread_width - net_premium_received

        results.append({
            'event': evt_dt.strftime('%Y-%m-%d'),
            'S0': S0,
            'S_exp': S_exp,
            'K1': K1,
            'K2': K2,
            'premium_received': net_premium_received,
            'payoff': payoff,
            'pl': pl,
            'capital_at_risk': capital_at_risk,
            'vol_estimate': vol_estimate,
            'price_move_pct': (S_exp - S0) / S0,
            'strike_otm_pct': (K1 - S0) / S0
        })

    return pd.DataFrame(results)

def calculate_benchmark_returns(results_df, price_data, benchmark_ticker='SPY'):
    try:
        start_date = price_data.index[0]
        end_date = price_data.index[-1]

        benchmark_data = yf.download(benchmark_ticker, start=start_date, end=end_date,
                                   progress=False, auto_adjust=False)
        if benchmark_data.empty:
            print(f"  Could not fetch {benchmark_ticker} data")
            return None

        benchmark_returns = []

        for _, trade in results_df.iterrows():
            event_date = pd.to_datetime(trade['event'])
            exp_date = event_date + pd.Timedelta(days=TRADING_PARAMS['days_to_expiration'])

            bench_prices = benchmark_data['Close']
            bench_start = bench_prices[bench_prices.index <= event_date]
            bench_end = bench_prices[bench_prices.index >= exp_date]

            if len(bench_start) > 0 and len(bench_end) > 0:
                start_price = bench_start.iloc[-1]
                end_price = bench_end.iloc[0]
                bench_return = (end_price - start_price) / start_price
                bench_dollar_return = bench_return * trade['S0']
                benchmark_returns.append(bench_dollar_return)

        return np.array(benchmark_returns)

    except Exception as e:
        print(f"  Error calculating benchmark returns: {e}")
        return None

def calculate_capm_alpha(strategy_returns, benchmark_returns):
    if len(strategy_returns) != len(benchmark_returns) or len(strategy_returns) < 5:
        return 0, 0, 0, 1.0, 1.0

    try:
        X = sm.add_constant(benchmark_returns)
        model = sm.OLS(strategy_returns, X).fit()

        alpha = model.params[0]
        beta = model.params[1]
        r_squared = model.rsquared
        alpha_pvalue = model.pvalues[0]
        beta_pvalue = model.pvalues[1]

        return alpha, beta, r_squared, alpha_pvalue, beta_pvalue

    except:
        return 0, 0, 0, 1.0, 1.0

def analyze_strategy_significance(results_df, price_data, symbol=SYMBOL, benchmark_ticker='SPY'):
    if results_df.empty or len(results_df) < 10:
        print("Insufficient data for significance testing")
        return

    returns = results_df['pl'].values
    win_rate = (returns > 0).mean()

    print("\n" + "="*60)
    print(f"REALISTIC STRATEGY ANALYSIS - {symbol}")
    print("="*60)

    if win_rate > 0.90:
        print(f"CRITICAL: Win rate {win_rate:.1%} > 90% - likely backtest error")
        return

    print(f"\n1. BASIC PERFORMANCE:")
    print(f"Trades: {len(returns)}")
    print(f"Win Rate: {win_rate:.1%}")
    print(f"Avg P/L: ${returns.mean():.2f}")
    print(f"Std Dev: ${returns.std():.2f}")
    print(f"Best Trade: ${returns.max():.2f}")
    print(f"Worst Trade: ${returns.min():.2f}")

    print(f"\n2. ALPHA DETECTION vs {benchmark_ticker}:")

    benchmark_returns = calculate_benchmark_returns(results_df, price_data, benchmark_ticker)

    if benchmark_returns is not None:
        strategy_total_return = returns.sum()
        benchmark_total_return = benchmark_returns.sum()

        strategy_avg_return = returns.mean()
        benchmark_avg_return = benchmark_returns.mean()

        print(f"Strategy Total Return: ${strategy_total_return:.2f}")
        print(f"Benchmark Total Return: ${benchmark_total_return:.2f}")
        print(f"Strategy Avg Return: ${strategy_avg_return:.2f}")
        print(f"Benchmark Avg Return: ${benchmark_avg_return:.2f}")

        alpha, beta, r_squared, alpha_pvalue, beta_pvalue = calculate_capm_alpha(returns, benchmark_returns)

        print(f"CAPM Regression Results:")
        print(f"  Alpha (excess return): {alpha:.4f} (p={alpha_pvalue:.4f})")
        print(f"  Beta (market exposure): {beta:.3f} (p={beta_pvalue:.4f})")
        print(f"  R-squared: {r_squared:.3f}")

        if alpha_pvalue < 0.05:
            if alpha > 0:
                print(f"  SIGNIFICANT POSITIVE alpha - strategy has genuine edge")
            else:
                print(f"  SIGNIFICANT NEGATIVE alpha - strategy underperforms market")
        else:
            print(f"  INSIGNIFICANT alpha - results could be random")

        if beta < 0.3:
            print(f"  LOW beta - strategy has low market correlation")
        elif beta > 0.7:
            print(f"  HIGH beta - returns may just be market exposure")

        if r_squared < 0.3:
            print(f"  LOW r-squared - strategy behavior differs from market")
        else:
            print(f"  HIGH r-squared - returns closely track market")

    print(f"\n3. REALISTIC ANNUALIZED METRICS:")

    avg_capital = results_df['capital_at_risk'].mean()
    avg_return = returns.mean()

    if avg_capital > 0:
        roc_per_trade = avg_return / avg_capital
    else:
        roc_per_trade = 0

    trades_per_year = 12
    risk_per_trade = 0.02
    realistic_annual_roc = roc_per_trade * risk_per_trade * trades_per_year

    print(f"Return on Capital per trade: {roc_per_trade:.1%}")
    print(f"Realistic Annual ROC: {realistic_annual_roc:.1%} (with 2% position sizing)")

    print(f"\n4. REALITY CHECKS:")

    issues = []
    if realistic_annual_roc > 0.50:
        issues.append(f"Annual ROC {realistic_annual_roc:.1%} > 50% - very high")
    elif realistic_annual_roc > 0.20:
        issues.append(f"Annual ROC {realistic_annual_roc:.1%} > 20% - aggressive but possible")

    if win_rate > 0.80:
        issues.append(f"Win rate {win_rate:.1%} > 80% - suspiciously high")
    if len(returns) < 20:
        issues.append(f"Only {len(returns)} trades - small sample")

    if 'alpha' in locals() and alpha > 0.1:
        issues.append(f"Very high alpha {alpha:.3f} - potentially overfitted")
    if 'r_squared' in locals() and r_squared > 0.8:
        issues.append(f"High r-squared {r_squared:.3f} - may just be market beta")

    if issues:
        for issue in issues:
            print(f"ISSUE: {issue}")
    else:
        print(f"REASONABLE: Results appear within realistic bounds")

    print(f"\n5. PRACTICAL CONSIDERATIONS:")
    required_account = avg_capital * 50
    print(f"Required account size: ${required_account:,.0f} (50x position sizing)")
    print(f"Return on account: {realistic_annual_roc:.1%} (with 2% position sizing)")

def generate_trading_instructions(results_df, symbol=SYMBOL):
    if results_df.empty:
        print("No results to generate trading instructions")
        return

    next_earnings = get_next_earnings_date(symbol)

    print("\n" + "="*70)
    print("TRADING INSTRUCTIONS FOR NEXT EARNINGS")
    print("="*70)

    print(f"\nNEXT EARNINGS DATE: {next_earnings}")

    recent_trades = results_df.tail(5)

def walk_forward_backtest(symbol=SYMBOL):
    global price_data, events, TRADING_PARAMS

    spread_width = TRADING_PARAMS['spread_width']
    days_to_exp = TRADING_PARAMS['days_to_expiration']
    min_history = TRADING_PARAMS['min_history_days']
    commission = TRADING_PARAMS['commission_per_contract']
    bid_ask_spread = TRADING_PARAMS['bid_ask_spread_pct']

    S = price_data['price']
    results = []

    print(f"Analyzing: {symbol}")
    print(f"Trading Parameters: ${spread_width} spread, {days_to_exp} DTE, "
          f"${commission} commission, {bid_ask_spread:.1%} spread cost")
    print(f"Events to match: {len(events)}")

    for evt in events:
        try:
            evt_dt = pd.to_datetime(evt)
        except:
            continue

        if evt_dt in S.index:
            idx = S.index.get_loc(evt_dt)
        else:
            days_diff = np.abs(S.index - evt_dt)
            closest_idx = days_diff.argmin()
            idx = closest_idx
            evt_dt = S.index[closest_idx]

        if idx < min_history or idx + days_to_exp >= len(S):
            continue

        hist = price_data.iloc[idx-min_history:idx]
        S0 = S.iloc[idx]
        S_exp = S.iloc[idx+days_to_exp]

        vol_estimate = np.sqrt(hist['returns'].var() * 252)
        vol_estimate = max(min(vol_estimate, 1.0), 0.15)

        K1 = round(S0)
        K2 = K1 + spread_width

        T = days_to_exp/252

        theoretical_premium = price_call_spread(S0, K1, K2, T, 0, vol_estimate)

        premium_after_costs = theoretical_premium * (1 - bid_ask_spread) - (2 * commission)

        min_acceptable_premium = 0.10
        strike_adjustment = 0

        while premium_after_costs < min_acceptable_premium and strike_adjustment < 10:
            strike_adjustment += 1
            K1 = round(S0 * (1 + 0.01 * strike_adjustment))
            K2 = K1 + spread_width
            theoretical_premium = price_call_spread(S0, K1, K2, T, 0, vol_estimate)
            premium_after_costs = theoretical_premium * (1 - bid_ask_spread) - (2 * commission)

        if premium_after_costs < min_acceptable_premium:
            continue

        premium_after_spread = max(theoretical_premium * (1 - bid_ask_spread), 0.01)
        commission_cost = 2 * commission
        net_premium_received = premium_after_spread - commission_cost

        payoff = min(max(S_exp - K1, 0), spread_width)
        pl = payoff - net_premium_received

        capital_at_risk = spread_width - net_premium_received

        results.append({
            'event': evt_dt.strftime('%Y-%m-%d'),
            'S0': S0,
            'S_exp': S_exp,
            'K1': K1,
            'K2': K2,
            'premium_received': net_premium_received,
            'payoff': payoff,
            'pl': pl,
            'capital_at_risk': capital_at_risk,
            'vol_estimate': vol_estimate,
            'price_move_pct': (S_exp - S0) / S0,
            'strike_otm_pct': (K1 - S0) / S0
        })

    df = pd.DataFrame(results)

    print(f"\nTRADE SUMMARY for {symbol}:")
    print(f"Trades executed: {len(df)}")

    if df.empty:
        print("No trades met basic criteria")
        return df

    total_ev = df['pl'].sum()
    avg_ev = df['pl'].mean()
    win_rate = (df['pl'] > 0).mean()
    avg_premium = df['premium_received'].mean()
    avg_capital = df['capital_at_risk'].mean()

    print(f"\nPERFORMANCE RESULTS:")
    print(f"Total EV: ${total_ev:.2f}")
    print(f"Avg EV/trade: ${avg_ev:.2f}")
    print(f"Win rate: {win_rate:.1%}")
    print(f"Avg premium received: ${avg_premium:.2f}")
    print(f"Avg capital at risk: ${avg_capital:.2f}")

    print(f"\nTRADE BREAKDOWN:")
    print(f"Stock went UP: {(df['price_move_pct'] > 0).sum()} trades")
    print(f"Stock went DOWN: {(df['price_move_pct'] < 0).sum()} trades")
    print(f"Stock FLAT: {(df['price_move_pct'] == 0).sum()} trades")
    print(f"Max price move: {df['price_move_pct'].max():.1%}")
    print(f"Min price move: {df['price_move_pct'].min():.1%}")

    plot_pl_density(df['pl'].values, symbol)
    analyze_strategy_significance(df, price_data, symbol)
    test_parameter_scenarios(symbol, df)
    generate_trading_instructions(df, symbol)

    return df
